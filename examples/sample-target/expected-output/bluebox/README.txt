AUTO OSCP RECON NOTES - WORKSPACE

This folder holds organized recon notes for one authorized lab target.

Layout:
  recon/        parsed scan data (services.json, nmap-summary.txt)
  notes/        your working notes (services.txt, web.txt, ...)
  checklists/   generated enumeration checklists
  report.txt    final plain-text report template

Reminder: this tool organizes recon output you provide. It does not scan
or exploit anything. Use it only against systems you are authorized to test.
